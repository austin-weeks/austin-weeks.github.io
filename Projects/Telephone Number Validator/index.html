How does this work?

The app receives the user's phone number input text, removes any white space, dashes, and parenthesis, and coverts any letters to their appropriate number values. The length of this number is then checked. If the length matches the standard length of a phone number with no country code, or the length of a number with a country code, and the first digit is 1 (the US country code), the number is valid!
